计算公式说明

基础（逐时刻）

  • 价差ₜ = 实时ₜ − 日前ₜ（仅当两点均有数值时计算）
  • 日前均 = AVG(日前ₜ)，实时均 = AVG(实时ₜ)，滚动均 = AVG(滚动ₜ)（仅统计非空）
  • 价差均 = AVG(价差ₜ)（仅统计价差非空时刻）
  • 实时>日前占比 = COUNT(价差ₜ>0) ÷ COUNT(价差ₜ非空)

全表顶部指标（跨工作表）

  • 全表日前均价 = AVG(各表日前均)
  • 全表实时均价 = AVG(各表实时均)
  • 全表平均价差 = AVG(各表价差均)
  • 平均实时>日前占比 = AVG(各表实时>日前占比)（按「天」等权，非按全部时刻点数加权）

逐日汇总 / 详情表

  • 日前P10/P50/P90：对当日日前电价做线性插值分位数（类似 Excel PERCENTILE.INC）
  • 日前波动 = STDEV.S(日前ₜ)（样本标准差，除以 n−1）
  • 滚动贴近：比较 AVG(|滚动−日前|) 与 AVG(|滚动−实时|)，差值 > 0.01 元/MWh 时判为更贴近一方，否则为「相当」
  • 极端时刻 Top10：按价差或日前价排序取前 10（全表所有时刻）

峰 / 平 / 谷（按时刻整点小时划分）

  • 谷段：0:00–7:59；峰段：9:00–11:59、17:00–21:59；平段：其余时刻
  • 各段均价 = 该段内有效点的算术平均（与 Excel 段内 AVERAGE 一致，前提是时刻划分一致）

说明

  • 数值内部保留 6 位小数计算，页面展示 4 位小数。
  • 某工作表行数不是 24 或存在重复时刻时，仍会参与统计，顶部会提示异常工作表。

分时电价曲线（日前 / 实时 / 滚动撮合）

纵轴为电价（元/MWh）。价差ₜ = 实时ₜ − 日前ₜ。三条线越分开，差异越明显。

分时价差（实时 − 日前）

价差ₜ = 实时ₜ − 日前ₜ；红色为正（实时更贵），绿色为负，柱子越高偏差越大。

分时段汇总（峰 / 平 / 谷）

段内 日前均/实时均/价差均/滚动均 均为该段非空点的算术平均；占比见下方文字。
